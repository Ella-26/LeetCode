### 分析

2 的幂在二进制中**有且只有一个 1**：

```
2⁰ = 1   → 0001
2¹ = 2   → 0010
2² = 4   → 0100
2³ = 8   → 1000
```

### 关键技巧：`n & (n - 1)`

- `n - 1` 会把 n **最右边的 1 变成 0，它后面的 0 全变 1**
- 如果 n 是 2 的幂（只有一个 1），两个数错位，`&` 结果必然是 0

```
n = 8:   1000
n-1=7:   0111
&:       0000 = 0  ✅

n = 6:   0110
n-1=5:   0101
&:       0100 ≠ 0  ❌
```

### 解法 1：循环除 2（直观）

```python
def isPowerOfTwo(self, n: int) -> bool:
    if n <= 0:
        return False
    while n % 2 == 0:
        n //= 2
    return n == 1
```

> 一直除以 2，除到底如果是 1 就是 2 的幂。

### 解法 2：位运算（一行）

```python
def isPowerOfTwo(self, n: int) -> bool:
    return n > 0 and (n & (n - 1)) == 0
```

> 时间复杂度 O(1)，面试加分。

### 注意

- `n <= 0` 直接返回 False（0 和负数都不是 2 的幂）
