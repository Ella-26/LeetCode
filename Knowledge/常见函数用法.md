## 一、循环思路

```
┌────────────────────────────────┐
│ 1. 要重复做吗？                │
│    不知道次数 → while          │
│    知道次数   → for            │
├────────────────────────────────┤
│ 2. 每一轮干什么？（口语说出来）  │
│    把这句话翻译成代码           │
├────────────────────────────────┤
│ 3. 啥时候停？（写出条件）       │
│    把它塞进 while/for 里       │
└────────────────────────────────┘
```

| 写法 | 代码 | 结果 |
|------|------|------|
| 固定次数 | `for i in range(n):` | `i` = 0, 1, ..., n-1 |
| 指定范围 | `for i in range(a, b):` | `i` = a, ..., b-1 |
| 指定步长 | `for i in range(a, b, step):` | 如 `range(0,10,2)` → 0,2,4,6,8 |
| 直接遍历数组 | `for num in arr:` | 每轮自动取下一个值 |

## 二、常用内置函数

| 函数 | 作用 | 示例 |

| `len(x)` | 取长度（字符串、列表等） | `len([1,2,3])` → `3` |

## 五、十进制 → 二进制（短除法）

不断除以 2，余数**从下往上**读。

```
例：6 转二进制
6 ÷ 2 = 3  余 0
3 ÷ 2 = 1  余 1
1 ÷ 2 = 0  余 1

结果：110 → 补齐 4 位，左边补 0：0110
```

## 六、位运算符

| 运算符 | 名称 | 规则 | 示例 |
|--------|------|------|------|
| `&` | 按位与 | 两边都是 1 才得 1 | `5 & 3` → `0101 & 0011` → `0001` → 1 |
| `\|` | 按位或 | 有一个为 1 就得 1 | `5 \| 3` → `0101 \| 0011` → `0111` → 7 |
| `^` | 按位异或 | 相同得 0，不同得 1 | `5 ^ 3` → `0101 ^ 0011` → `0110` → 6 |
| `~` | 按位取反 | 0 变 1，1 变 0 | `~5` → `-(5+1)` → -6 |
| `<<` | 左移 | 整体左移，右边补 0 | `1 << 3` → 8 |
| `>>` | 右移 | 整体右移，左边补 0 | `8 >> 2` → 2 |

**公式：** `x << n = x × 2ⁿ` `x >> n = x // 2ⁿ`

**`~x` 口诀：** `~x = -(x + 1)`，如 `~5` = -6, `~0` = -1, `~(-1)` = 0

## 七、位运算常见套路

| 操作 | 写法 | 示例 |
|------|------|------|
| 判断奇偶 | `n & 1` | `3 & 1` → 1（奇），`4 & 1` → 0（偶） |
| 判断 2 的幂 | `n > 0 and n & (n - 1) == 0` | `8` → ✅，`6` → ❌ |
| 乘 2 / 除 2 | `n << 1` / `n >> 1` | `5 << 1` → 10，`6 >> 1` → 3 |
| 交换两数（无临时变量） | `a ^= b; b ^= a; a ^= b` | |
| 取第 k 位 | `(n >> k) & 1` | `(5 >> 2) & 1` → 1 |
| 第 k 位设为 1 | `n \| (1 << k)` | `5 \| (1 << 1)` → 7 |
| 第 k 位设为 0 | `n & ~(1 << k)` | `5 & ~(1 << 0)` → 4 |
| 统计 1 的个数 | `bin(n).count('1')` | `bin(6).count('1')` → 2 |

## 八、列表（数组）用法

### 1. 创建

| 操作 | 写法 | 结果 |
|------|------|------|
| 空列表 | `arr = []` | `[]` |
| 定长全 0 | `arr = [0] * 5` | `[0, 0, 0, 0, 0]` |
| 直接初始化 | `arr = [1, 2, 3]` | `[1, 2, 3]` |

### 2. 基本操作

| 操作 | 写法 | 对比 Java |
|------|------|-----------|
| 长度 | `len(arr)` | `arr.length` |
| 取元素 | `arr[i]` | `arr[i]` |
| 改元素 | `arr[i] = 5` | `arr[i] = 5` |
| 取最后一个 | `arr[-1]` | `arr[arr.length-1]` |
| 取切片 [1,4) | `arr[1:4]` | 无 |
| 判断元素在不在 | `5 in arr` | 无 |
| 反转 | `arr[::-1]` | 无 |
| 排序（原地） | `arr.sort()` | `Arrays.sort(arr)` |
| 排序（新列表） | `sorted(arr)` | 无 |

### 3. 添加 / 删除

| 操作 | 写法 | 说明 |
|------|------|------|
| 末尾加 | `arr.append(val)` | `[1,2]` → `[1,2,val]` |
| 指定位置插 | `arr.insert(i, val)` | 索引 i 前插入 |
| 删最后一个 | `arr.pop()` | 返回被删值 |
| 删指定位置 | `arr.pop(i)` | 删索引 i |
| 按值删除 | `arr.remove(val)` | 删第一个匹配的 |

### 4. 遍历

| 方式 | 写法 |
|------|------|
| 按索引 | `for i in range(len(arr)):` |
| 直接取值 | `for num in arr:` |
| 索引 + 值 | `for i, num in enumerate(arr):` |

### 5. 二维列表

| 操作 | 写法 |
|------|------|
| 创建 rows×cols 全 0 | `[[0] * cols for _ in range(rows)]` |
| ❌ 错误写法 | `[[0] * cols] * rows`（所有行指向同一列表！） |
| 行数 / 列数 | `len(matrix)` / `len(matrix[0])` |
| 遍历 | `for i in range(rows): for j in range(cols):` |
