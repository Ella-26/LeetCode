## 一、循环思路

```
┌────────────────────────────────┐
│ 1. 要重复做吗？                │
│    不知道次数 → while          │
│    知道次数   → for            │
├────────────────────────────────┤
│ 2. 每一轮干什么？（口语说出来）  │
│    把这句话翻译成代码           │
├────────────────────────────────┤
│ 3. 啥时候停？（写出条件）       │
│    把它塞进 while/for 里       │
└────────────────────────────────┘
```

## 五、十进制 → 二进制（短除法）

不断除以 2，余数**从下往上**读。

```
例：6 转二进制
6 ÷ 2 = 3  余 0
3 ÷ 2 = 1  余 1
1 ÷ 2 = 0  余 1

结果：110 → 补齐 4 位，左边补 0：0110
```

## 六、六个位运算符

位运算符不看数字的值，而是看它**底层的二进制位**。

| 运算符 | 名称 | 规则 | 示例 |
|--------|------|------|------|
| `&` | 按位与 | 两边都是 1 才得 1 | `5 & 3` = `0101 & 0011` = `0001` = 1 |
| `\|` | 按位或 | 有一个为 1 就得 1 | `5 \| 3` = `0101 \| 0011` = `0111` = 7 |
| `^` | 按位异或 | 相同得 0，不同得 1 | `5 ^ 3` = `0101 ^ 0011` = `0110` = 6 |
| `~` | 按位取反 | 0 变 1，1 变 0 | `~5` = `-(5+1)` = -6 |
| `<<` | 左移 | 整体左移，右边补 0 | `1 << 3` = 8 |
| `>>` | 右移 | 整体右移，左边补 0 | `8 >> 2` = 2 |

### 左右移公式

```python
x << n   =   x × 2ⁿ     # 左移 = 乘 2 的 n 次方
x >> n   =   x // 2ⁿ    # 右移 = 整除 2 的 n 次方
```

### 详解 `~` 取反

```python
~x = -(x + 1)

~5  = -6
~0  = -1
~-1 = 0
```

## 七、位运算常见套路速查

| 操作 | 写法 | 示例 |
|------|------|------|
| 判断奇偶 | `n & 1` | `3 & 1` → 1（奇），`4 & 1` → 0（偶） |
| 判断 2 的幂 | `n > 0 and n & (n - 1) == 0` | `8` → ✅，`6` → ❌ |
| 乘以 2 | `n << 1` | `5 << 1` → 10 |
| 除以 2 | `n >> 1` | `6 >> 1` → 3 |
| 交换两数（无临时变量） | `a ^= b; b ^= a; a ^= b` | |
| 取第 k 位 | `(n >> k) & 1` | `(5 >> 2) & 1` → 1 |
| 第 k 位设为 1 | `n \| (1 << k)` | `5 \| (1 << 1)` → 7 |
| 第 k 位设为 0 | `n & ~(1 << k)` | `5 & ~(1 << 0)` → 4 |
| 统计 1 的个数 | `bin(n).count('1')` | `bin(6).count('1')` → 2 |
